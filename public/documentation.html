Movie Web Application named "MovieMemento"

The application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies. Users might also be able to create a "To watch" list

Documentation of API endpoints
URL Method Request data Response data Description
/movies GET -- A Json object holding data about all movies on the app, structured like this:

{
Title: "",
OriginalTitle: "",
Description: "";
Genre: {
Name: "",
Description: ""
},
Director: {
Name: "",
Bio: "",
Birthyear: ""
},
imageURL: ""
}

 Returns a list of all movies to the user
/movies/:title GET -- A Json object holding data about a specific movie. Containing the title, genre, director, movie description, imageURL. Returns data about a single movie by title
/movies/genre/:Name GET -- A Json object holding data about a genre title and its description Returns data about a genre (description) by name/title (e.g., “Thriller”)
/movies/director/:Name GET -- A Json object holding data about a single director, containing their movies, name, biografie and birth year Return data about a director (movies, name, bio, birth year) by name
/users POST A Json object holding data about the new user to be added, structured like this:

{
"Username": "String",
"Password": "String",
"Email": "String",
"Birthday": "Date"
}

A Json object holding data about the new user that was added, including a generated ID. Example:

{
Username: Toni,
Password: 1234,
Email: MyEmail@toni.com,
Birthday: 1900-12-12,
FavoriteMovies: [],
_id: 42
}

Allows new users to register
/users/:Username PUT A Json object holding data about the username thats gonna be updated. Structured like this:

{
Username: String,
Password: String,
Email: String,
Birthday: Date
}

A Json object showing the updated data about the user. Example:

{
_id: 42
Username: NewUserName,
Password: 1234,
Email: NewEmail@toni.com,
Birthday: 1900-12-12,
FavoriteMovies: []
}

Allows users to update their user info
/users/:Username/movies/:MovieID POST -- A Json object showing the users profil with the added favorite movie. Example:

{
_id: 42
Username: NewUserName,
Password: 1234,
Email: NewEmail@toni.com,
Birthday: 1900-12-12,
FavoriteMovies: [ "6939845300d65a05931e2628" ]
}

Allows users to add a movie to their list of favorite movies
/users/:Username/movies/:MovieID DELETE -- A Json object showing the users profile, showing that one of the favorite movies has been deleted from the array. Example:

{
_id: 42
Username: NewUserName,
Password: 1234,
Email: NewEmail@toni.com,
Birthday: 1900-12-12,
FavoriteMovies: []
}

Allows users to remove a movie from their list of favorite movies
/users/:Username DELETE -- Showing message that a user has been removed Allows existing users to deregister
