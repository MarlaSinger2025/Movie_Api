Movie Web Application named "MovieMemento"

The application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies. Users might also be able to create a "To watch" list

Documentation of API endpoints
URL Method Request data Response data Description
/movies GET -- A Json object holding data about all movies on the app, structured like this:

{
Title: "",
OriginalTitle: "",
Description: "";
Genre: {
Name: "",
Description: ""
},
Director: {
Name: "",
Bio: "",
Birthyear: ""
},
imageURL: ""
}

 Returns a list of all movies to the user
/movies/:title GET -- A Json object holding data about a specific movie. Containing the title, genre, director, movie description, imageURL. Returns data about a single movie by title
/movies/genre/:genreName GET -- A Json object holding data about a genre title and its description Returns data about a genre (description) by name/title (e.g., “Thriller”)
/movies/director/:directorName GET -- A Json object holding data about a single director, containing name, biografie and birth year Return data about a director (name, bio, birth year) by name
/users POST A Json object holding data about the new user to be added, structured like this:

{
Name: "",
Favorites: []
}

A Json object holding data about the new user that was added, including a generated ID. Example:

{
Name: "Toni",
Favorites: ["Amelie", "Labyrinth"],
ID: "1"
}

Allows new users to register
/users/:id PUT A Json object holding data about the username thats gonna be updated. Structured like this:

{
"Name": "[Enter new name]",
"Favorites": []
}

Showing a message that the username was successfully updated Allows users to update their user info (username)
/users/:id/:movieTitle POST -- Showing a message that the a movie was successfully added to the favorites list Allows users to add a movie to their list of favorite movies
/users/:id/:movieTitle DELETE -- Showing a message that a movie has been successfully removed from the favorites list Allows users to remove a movie from their list of favorite movies
/users/:id DELETE -- Showing message that a user has been removed Allows existing users to deregister
